# Elissa 8 Bit Chess Set

Inspired by the 1877 tall ship Elissa at Galveston Historic Seaport.

## Piece Mapping

* King: ship captain at the wheel
* Queen: Elissa as a three masted barque
* Bishop: mast, sail, and rigging
* Knight: iron hull bow and figurehead
* Rook: iron anchor and teak capstan
* Pawn: deckhand carrying coiled rope

## Files

The individual transparent PNG files follow Khon's Chess theme naming:

* `white-k.png`
* `white-q.png`
* `white-b.png`
* `white-n.png`
* `white-r.png`
* `white-p.png`
* `black-k.png`
* `black-q.png`
* `black-b.png`
* `black-n.png`
* `black-r.png`
* `black-p.png`

Place the folder inside `pieces/elissa/` and register `elissa` in the theme selector and theme loading logic.

The included `elissa-sprite-sheet.png` is a transparent preview of the complete set.
